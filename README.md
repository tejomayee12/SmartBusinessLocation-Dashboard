# SmartBusinessLocation-Dashboard
📊 Business Location Performance Dashboard (Excel | Power BI Concepts)
🧠 Project Overview

An interactive business intelligence dashboard built in Microsoft Excel to analyze multi-country retail performance using Power Query, Power Pivot, and DAX.

The dashboard provides strategic insights into revenue trends, customer behavior, and product performance across different locations.

🎯 Business Objective

To help stakeholders:

Identify top-performing countries

Analyze gender-based purchasing behavior

Compare year-over-year performance

Monitor KPI growth

Track customer recency and engagement

📌 Key Performance Indicators (KPIs)

💰 Total Revenue

👥 Total Customers

🧾 Transactions

📦 Average Order Value (AOV)

⏳ Recency Analysis

📊 Dashboard Features

✔ Interactive slicers (Country, Gender, Year)
✔ Revenue Map (Geographical Performance)
✔ Gender Breakdown by Country
✔ Last Year Trend Analysis (Qty & Amount)
✔ 2-Year Purchase Comparison
✔ Popular Country Analysis
✔ Dynamic KPI Cards

🛠 Technical Stack

Microsoft Excel

Power Query (Data Cleaning & Transformation)

Power Pivot (Data Modeling)

DAX Measures

Pivot Tables & Charts

Map Visualization
